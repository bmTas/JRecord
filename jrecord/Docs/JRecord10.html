Record Editor

This packages IO routines and line / field access code was originally written for the Record-Editor. If you also have the Record-Editor. you can:

  1. Import Record-Layout's into the Record-Editor (LayoutEditor - Import Function).
  2. Edit a Record-Layout via screens
  3. Export the Record-Layout back out as a file (LayoutEditor - Copy Function).
